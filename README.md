#### Modo de Desenvolvimento

Instale os pacotes necessários

```
yarn
```

Subir o servidor de desenvolvimento

```
yarn dev
```

Build a versão final

- será gerado uma pasta com o nome 'dist'
- dist é onde os arquivos estão prontos para deploy

```
yarn build
```
